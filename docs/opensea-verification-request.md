# OpenSea Verification Request — THE 1969

## Where to submit

Three valid paths, in order of speed:

1. **OpenSea Studio** (fastest if it's a Drop): collection page → Settings → Verification → "Apply for verification". This is the in-app flow for OpenSea Drops collections and is usually 24-72h.
2. **OpenSea support form**: https://support.opensea.io → submit a ticket → "Account & verification" → "Apply for collection verification". Paste the body below.
3. **Direct DM** to `@opensea` and `@OpenSeaSupport` on X with a short version of the body, the collection link, and a request for verification consideration. Useful for visibility, not the official path.

## Tips that have worked for other collections

- **Volume threshold** is the unofficial gate (~75 ETH cumulative).
  If you're not there yet, wait until you are. Sold-out + low volume
  reads as a friends-and-family mint to OpenSea reviewers.
- **Etherscan-verified contract** counts. We have that.
- **Real utility** counts more in 2026 than it did in 2022. The
  staking vault is your ace — make sure they see it.
- **Don't spam the form**. One clean submission, then wait 72h before
  any follow-up.
